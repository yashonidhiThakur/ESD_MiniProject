This is a Spring Boot application for managing student timetables and courses. The system allows students to view and download their timetable, and handle authentication. It also provides functionalities to manage students, courses, and related data with role-based security.

Backend part of the project.

Prerequisites-
Install dependencies using Maven:
mvn install
Start the Spring Boot application:
mvn spring-boot:run
Access the application by navigating to http://localhost:30000 in your browser.
